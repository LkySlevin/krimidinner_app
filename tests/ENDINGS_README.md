# Schluss-Szenen Übersicht


<details>
<summary><strong>Logikbaum der Enden</strong></summary>

- Urteil korrekt (Mörder erwischt)
  - Liebender angeklagt? → Partner bricht beim Schuss zusammen
  - sonst: beide Liebenden leben → schleichen zusammen aufs Zimmer
  - Intrigant: Ziel getroffen? → zufrieden-grinsendes Gute Nacht; sonst Humbug-Kommentar vor Schlussabsatz
  - Verzweifelte Person: nicht möglich als Mörder
- Urteil falsch (Unschuldiger gehängt)
  - Liebender angeklagt? → Partner bricht beim Strick zusammen
  - sonst: beide Liebenden leben → schleichen zusammen aufs Zimmer
  - Intrigant: Ziel gehängt? → zufrieden-grinsendes Gute Nacht; sonst Humbug-Kommentar vor Schlussabsatz
  - Verzweifelte Person gehängt? → lächelnd; sonst geht sie mit hängendem Kopf ins Zimmer
</details>


| Szenario | Urteil korrekt? | Besonderheiten |
| --- | --- | --- |
| Richtiges Urteil – Liebender stirbt | Ja | Liebespaar, Verzweifelte Person, Intrigant |
| Richtiges Urteil – Liebespaar überlebt | Ja | Liebespaar, Verzweifelte Person, Intrigant |
| Richtiges Urteil – Intrigant erfolgreich | Ja | Liebespaar, Verzweifelte Person, Intrigant |
| Richtiges Urteil – Intrigant scheitert | Ja | Liebespaar, Verzweifelte Person, Intrigant |
| Falsches Urteil – Liebender stirbt | Nein | Liebespaar, Verzweifelte Person, Intrigant |
| Falsches Urteil – Liebespaar überlebt | Nein | Liebespaar, Verzweifelte Person, Intrigant |
| Falsches Urteil – Verzweifelte Person stirbt | Nein | Liebespaar, Verzweifelte Person, Intrigant |
| Falsches Urteil – Verzweifelte Person überlebt | Nein | Liebespaar, Verzweifelte Person, Intrigant |
| Falsches Urteil – Intrigant erfolgreich | Nein | Liebespaar, Verzweifelte Person, Intrigant |
